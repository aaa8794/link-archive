# archiv*o PRD
> 흘려보내지 않는 영감 포착 도구 — 관찰에서 아이디어로

---

## Overview

영감은 순간적이다. 지나치면 영원히 잊혀진다.

archiv*o는 일상 속 영감을 빠르게 포착하고, 쌓인 영감들이 아이디어로 연결되는 경험을 만든다.
핵심 플로우: **관찰 → 영감 발견 → 영감 결합**

---

## Problem

- 영감을 저장해도 **왜 저장했는지 맥락이 사라진다**
- 저장한 것들이 레퍼런스 폴더에 방치되고 꺼내 쓰이지 않는다
- 주니어 창작자는 영감을 수집하는 **습관 자체가 없다**
- 기존 도구(핀터레스트, Notion 등)는 너무 무겁거나 맥락이 없다

---

## Target Users

- **Primary**: 주니어 디자이너/기획자 (22~28세, 경력 0~3년)
- **Secondary**: 시니어 창작자, 누구든 창작하는 사람

---

## Core Concept

저장하는 순간의 마찰을 최소화한다.
저장 시 **"나의 한 마디"** 메모를 붙여 맥락을 보존한다.
미션을 통해 오프라인 관찰 습관을 자연스럽게 만든다.

---

## MVP Features

### 🔴 Must Have

#### 1. 빠른 저장
- 사진 / 텍스트 / 링크를 3초 안에 저장
- 저장 시 "나의 한 마디" 메모 첨부 가능
- 홈 화면에서 바로 진입 가능한 단축 버튼

#### 2. 오늘의 미션
- AI가 매일 관찰 미션 생성
- 미션 예시: "오늘 밖에서 노란 컬러를 3개 찾아보세요", "오늘 이슈 3개 찾기"
- 미션 완료 시 영감 저장으로 자연스럽게 연결
- 계절/날씨 기반 미션 생성 고려

#### 3. 영감 아카이브
- 저장된 영감을 시간순 / 태그별로 탐색
- 카드 형태 시각화
- 사진, 텍스트, 링크 혼합 표시

#### 4. 온보딩
- 서비스 철학 전달 + 첫 미션 안내
- 3단계 이내로 구성

### 🟡 Should Have

#### 5. 태그/분류
- 저장 시 무드/카테고리 태그 선택
- 자동 추천 + 직접 입력 가능

#### 6. 링크 아카이브
- URL 저장 시 OG 이미지/제목 자동 추출
- **요약 기능은 제외** (영감의 핵심은 느낌이지 내용 요약이 아님)

### 🟢 Nice to Have (V2)

#### 7. 영감 연결
- 저장된 영감들 사이 AI가 연결고리 제안
- "이것들 합치면 이런 아이디어 어때?" 형태

## Design System

- **Primary Color**: Archivo Blue (`#3B3EED`)
- **Symbol**: 별(*)이 O 안으로 쏙 들어가는 모션
- **톤앤매너**: 가볍고 빠름, 과하지 않음
- **컬러 팔레트**: Archivo 100~10 (딥블루~연블루), Black 100~White, Yellow 계열
