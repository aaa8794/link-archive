# Link Images Smoke Check

이 체크리스트는 Supabase 설정을 적용한 직후 실제 기능이 붙었는지 빠르게 확인하는 용도입니다.

## Supabase 적용 전제

- `public.links.images` 컬럼 생성 완료
- `link-images` bucket 생성 완료
- storage policy 적용 완료

## 로컬 실행

```bash
npm install
npm start
```

## 확인 순서

### 1. 이미지 없이 링크 추가

- 링크 추가 모달 열기
- URL, 제목만 입력
- 저장 성공 확인
- 상세 페이지에서 이미지 placeholder 유지 확인

### 2. 이미지 URL 추가

- 링크 추가 모달 열기
- 이미지 URL 1개 추가
- 미리보기 썸네일 생성 확인
- 저장 후 상세 페이지에서 캐러셀 1장 확인
- 점 인디케이터 1개 확인

### 3. 여러 이미지 URL 추가

- 이미지 URL 2개 이상 추가
- 추가 순서대로 미리보기 쌓이는지 확인
- 특정 이미지 삭제 후 순서가 자연스럽게 유지되는지 확인
- 저장 후 상세 페이지에서 좌우 이동과 점 인디케이터 개수 확인

### 4. 로컬 파일 업로드

- 로그인된 상태에서 링크 추가 모달 열기
- 이미지 파일 여러 장 선택
- 업로드 완료 후 미리보기 썸네일 확인
- 저장 후 상세 페이지 캐러셀에 업로드된 순서대로 표시되는지 확인

### 5. 비로그인 업로드 가드

- 로그인되지 않은 상태에서 파일 업로드 시도
- `로그인 후 이미지 업로드를 사용할 수 있어요.` 메시지 표시 확인

### 6. 인사이트 모션

- 상세 페이지 진입
- 상단 그라데이션 영역이 기본 상태에서 아주 미세하게 움직이는지 확인
- 인사이트 저장
- 저장 직후 그라데이션 필드가 더 또렷하게 섞이는 모션을 보이는지 확인
- 인사이트 개수 증가에 따라 색 조합 단계가 변하는지 확인
